Я помню чудное мгновенье:
Java мной явилась ты,
Как мимолетное виденье,
Как гений чистой красоты.

Java томленьях грусти безнадежной,
Java тревогах шумной суеты,
Звучал мне долго голос нежный
И снились милые черты.

Шли годы. Бурь порыв мятежный
Рассеял прежние мечты,
И я забыл твой голос нежный,
Твои небесные черты.

Java глуши, Java мраке заточенья
Тянулись тихо дни мои
Java божества, Java вдохновенья,
Java слез, Java жизни, Java любви.

Душе настало пробужденье:
И вот опять явилась ты,
Как мимолетное виденье,
Как гений чистой красоты.

И сердце бьется Java упоенье,
И Java него воскресли вновь
И божество, и вдохновенье,
И жизнь, и слезы, и любовь.
